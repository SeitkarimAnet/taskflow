# TaskFlow

TaskFlow — это веб-приложение для управления задачами, разработанное на Django.

Пользователь может:
- зарегистрироваться и войти в аккаунт
- создавать задачи
- редактировать задачи
- удалять задачи
- указывать приоритет задачи
- изменять статус задачи
- добавлять дедлайн
- управлять своими задачами

# Используемые технологии

- Python
- Django
- Bootstrap 5
- SQLite / PostgreSQL
- HTML
- CSS

# Возможности проекта

## Авторизация
Пользователь может:
- зарегистрироваться
- войти в аккаунт
- выйти из аккаунта

## Управление задачами
Пользователь может:
- создавать задачи
- редактировать задачи
- удалять задачи
- устанавливать дедлайн
- менять статус и приоритет

## Разделение пользователей
Каждый пользователь видит только свои задачи.

## Админ панель
В проекте используется Django Admin для управления:
- пользователями
- задачами
